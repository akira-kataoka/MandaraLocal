MandaraLocal 使い方

1. 起動

Windowsなら start.bat をダブルクリック。Mac/Linuxなら python3 scripts/serve.py。
ブラウザで http://localhost:8765 が自動で開きます。

2. データを読み込む

左サイドバー「データ読み込み」から:

  • サンプルデータを読み込む — 2020年国勢調査ベースの人口・面積・高齢化率データ
  • CSVファイルを開く — 自分のCSVをアップロード

3. CSV形式

列意味例
1列目都道府県名 or コード(1〜47)東京都, 13, Tokyo
2列目以降数値データ（複数列OK）14047594
都道府県,人口,面積
北海道,5224614,78421
青森県,1237984,9646
...
カンマ区切り、UTF-8推奨。空欄・「-」・「NA」は欠損として扱われ、地図上ではグレー表示されます。

4. 階級区分の選び方

方法特徴向くケース
等量区分（分位数）各クラスの件数がほぼ同じ分布の偏りに左右されず比較したい
等値区分（等間隔）値の幅で等分絶対値の差を視覚化したい
自然区分（Jenks）クラス内分散最小化データのまとまり（自然な切れ目）を尊重

5. 色パレットの使い分け

  • YlOrRd / Blues / Greens 等 — 連続的（小→大）。一般的な分布データに。
  • RdYlBu / Spectral — 発散型。基準値（平均、ゼロ等）からの上下を示すデータに。
  • Viridis — 色覚多様性対応。プレゼン資料に推奨。

6. エクスポート

  • PNG出力 — 現在の地図画面をそのまま画像に。ベースタイル付き。
  • SVG出力 — 都道府県境界のみ。Illustrator / Inkscapeで再編集可。

7. WEB公開

このディレクトリ全体をそのまま静的サイトホスティングにアップロードするだけです。
GitHub Pages の場合、リポジトリの Settings → Pages → ブランチを main に設定すれば公開完了。
