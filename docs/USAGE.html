MandaraNext 使い方

公開URL: https://akira-kataoka.github.io/MandaraNext/

1. 起動

Webから (推奨)

上記URLにアクセスするだけ。PWA対応なので 📲 アプリ化 ボタンでデスクトップ/ホーム画面に追加してオフラインでも使えます。

ローカルから

cd MandaraLocal
python scripts/serve.py
# http://localhost:8765/ が開く

3. データの読み込み

サンプル

「サンプルデータを読み込む」ボタン。レベルに応じて適切なCSVが読まれます。

自前データ

CSV / Excel (.xlsx, .xls) どちらでも対応。1列目=地域、2列目以降=数値。

テンプレートDL

「テンプレートCSVをDL」で 47行 or 1,742行の空テンプレが落ちます。Excelで開いて数値だけ入れて再読込。

追加マージ

「＋ 追加CSV (マージ)」で別のCSV/Excelを同じ地域キーで結合可能。複数指標を組み合わせた多変量分析に。

5. 階級区分の方法

  • 等量区分（分位数） — 各クラスの件数を均等に
  • 等値区分（等間隔） — 値の幅で等分
  • 自然区分（Jenks） — クラス内分散最小化
  • 手動区分 — 任意の境界値をカンマ区切り入力
  • 対数スケール — 歪んだ分布の見栄えを改善

6. 分析機能

  • 基本統計 — 平均/中央値/Q1/Q3/IQR/標準偏差/CV/歪度/尖度/最頻値
  • 外れ値検出 — IQR×1.5、上方/下方リスト
  • 派生列 — A÷B などその場で算出
  • 属性検索 — AND/OR/NOT を組み合わせた条件絞り込み
  • 距離測定 — 2点クリックで大圏距離
  • 面積測定 — 多角形クリック→ダブルクリックで確定
  • 空間検索（バッファ） — クリック点+半径で内側強調
  • 標準偏差楕円 — 点分布の方向性可視化
  • クロス集計 — 2列の頻度マトリクス
  • ヒストグラム — 分布の形状
  • 散布図 — ピアソンr・R²・回帰直線・地図と色同期
  • 時系列再生 — 年次データを自動アニメーション
  • 時空間派生 — 2時点の差分/増減率を派生列に

7. インタラクション

5つの双方向クロスハイライト：

  1. 凡例ホバー → 地図の同階級地域
  2. 散布図ホバー/クリック → 地図 (ズーム可)
  3. データ表 行ホバー → 地図
  4. ヒストグラム ホバー → 地図 (同bin)
  5. クロス集計セル ホバー → 地図 (rowBin×colBin)

地図クリックで詳細パネル。Shift+クリックで複数ピン留め。

9. 出力（9形式）

  • PNG — 地図全体 / 散布図単体 / ヒストグラム単体
  • SVG — タイトル+凡例つきベクター
  • PDF — A4横向き
  • アニメーションGIF — 時系列をループ化
  • KML — Google Earth で開く
  • GeoJSON — QGIS/ArcGIS用 (派生列も含む)
  • CSV — データ全体 / テンプレート
  • シーンJSON — 設定をファイルで共有
  • URL共有 — 設定をリンクで共有 (🔗ボタン)

10. シーン (設定の保存)

  • 💾 保存 — 現在の設定を名前付きで localStorage に保存
  • セレクタ — サンプル4種 + マイシーン
  • ⬇ エクスポート / ⬆ インポート — チーム間で配布
  • 🔗 URL共有 — クリップボードにリンクコピー

11. 編集ツール

  • 住所検索 — 国土地理院 API で住所→地図ピン
  • 市町村検索 — 市町村レベルで日本語/カナ/英語 部分一致
  • メッシュ生成 — JIS X 0410 1次/2次/3次
  • ✏ 描画 — ポリゴン/ライン/円/マーカーを自由に描いて GeoJSON 出力

12. ヒント

対象列の隣に「派生列を作る」を開けば 「人口÷面積=人口密度」のような計算列を1秒で追加。地図/散布図/ヒストグラム すべてで利用可。
主題図モードを「カートグラム」にすると、人口の多い県が原型のまま巨大化する変形地図に。プレゼン映え抜群。
町丁目モードは初回のみ Geolonia API への通信が必要。一度ロードすればキャッシュされます。
